How to analyze a chess position with Stockfish and Python

Posted by Tim Lewis on April 16, 2022
This tutorial illustrates how to:
  • analyze a chess position with Stockfish and Python.
  • use the Pystockfish to read a FEN string and provide a best move.



  1. Prequisite: Python is installed and working (preferrably on Linux):
    2. python --version
  2. Install stockfish executable:
    3.         
#To install on Linux Mint and Ubuntu machines
apt-get update -y 
apt-get install -y apt-utils stockfish
  3. Install pystockfish module:
    4. pip install pystockfish
  4. Use pystockfish to compute the best move from a fen string
    5.   
import pystockfish
#fen string for new game
fen = 'rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR w KQkq - 0 1'

#create Engine object and compute best move, based on the fen string
deep = pystockfish.Engine(depth=20)
deep.setfenposition(fen)
best_move = deep.bestmove()['move']
print(best_move)

    Save this code in a script called analyze_chess_game.py and then run it with Python.

    python analyze_chess_game.py
